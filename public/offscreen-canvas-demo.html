💡 什么是离屏渲染？

离屏渲染（OffscreenCanvas）是将 Canvas 渲染工作从主线程转移到 Web Worker 中执行的技术。 这样可以避免渲染阻塞主线程，提升页面流畅度。

  • 主线程渲染：渲染工作在主线程执行，可能阻塞 UI 交互
  • Worker 渲染：渲染工作在 Worker 线程执行，不影响主线程
  • 性能对比：弹幕越多，离屏渲染的优势越明显

🚀 如何使用？

1. 点击"开始渲染"按钮启动动画
2. 点击"添加 100 条弹幕"增加弹幕数量
3. 观察两种渲染方式的 FPS 和渲染时间差异
4. 尝试添加大量弹幕（1000+ 条）查看性能差异
