## 🚀 启动项目

### 1. 确保依赖已安装
```bash
cd d:\前端项目\webrtc
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器访问
```
http://localhost:5173
```

## 📱 页面路由导航

### 通过底部导航栏访问（主要页面）
```
http://localhost:5173/#/         - 首页
http://localhost:5173/#/discover - 发现
http://localhost:5173/#/follow   - 关注
http://localhost:5173/#/profile  - 我的
```

### 其他页面直接访问
```
http://localhost:5173/#/search      - 搜索
http://localhost:5173/#/message     - 消息
http://localhost:5173/#/view        - 直播间
http://localhost:5173/#/start-live  - 开始直播
```

## 🔄 页面跳转关系

```
首页 (/)
  ↓ 点击直播卡片
直播间 (/view)
  ↓ 点击主播头像
个人中心 (/profile)

发现页 (/discover)
  ↓ 点击搜索
搜索页 (/search)
  ↓ 搜索结果点击
直播间 (/view)

关注页 (/follow)
  ↓ 点击直播中的主播
直播间 (/view)

底部导航中间按钮
  ↓
开始直播 (/start-live)
```

## 🐛 已知问题

1. ⚠️ 所有 `script setup` 部分都是空的，需要根据实际需求补充
2. ⚠️ 图片使用的是占位符URL（pravatar.cc, picsum.photos）
